## VMware Notes vs. Device Notes

Notes added to VMware virtual devices outside of Device42 appear in the **Hypervisor Description** field on the device view and edit pages. If the VMware virtual device does not have notes, the field does not appear.

<ThemedImage
  alt="Hypervisor description field"
  sources={{
    light: useBaseUrl('/assets/images/device-virtual-blade/hypervisor-description-light.png'),
    dark: useBaseUrl('/assets/images/device-virtual-blade/hypervisor-description-dark.png'),
  }}
/>

The **Hypervisor Description** is separate from the Device42 **Notes** field that you use to add or edit notes within Device42.

<ThemedImage
  alt="Device42 Notes field"

## Blade Device Subtype

**Blade** is one of the physical device subtypes. You can also create custom subtypes as needed.

<ThemedImage
  alt="Blade subtype example"
  sources={{

    dark: useBaseUrl('/assets/images/device-virtual-blade/blade-subtype-dark.png'),
  }}
/>

You can set the **Device Host Chassis** for this blade and the **Blade Slot #**. The other fields are generally shared with other device subtypes.
